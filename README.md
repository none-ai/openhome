# OpenHome | 探索AI与技术的无限可能

> 深入探索 AI、Agent、大模型与前沿技术的研究博客

## 🚀 关于

这是一个使用 [Astro](https://astro.build) 构建的高性能静态博客，专注于分享 AI、Agent、大模型以及前沿技术的研究与实践。

## 📚 博客主题

- 🤖 **AI Agent**: 架构设计与实现
- 🧠 **大语言模型**: 应用实践与研究
- 🔧 **OpenClaw**: 自动化工具研究
- 📖 **技术笔记**: 最佳实践与经验分享

## 🛠 技术栈

- **框架**: [Astro](https://astro.build)
- **部署**: GitHub Pages
- **主题**: 自定义
- **内容**: Markdown / MDX

## 📦 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
